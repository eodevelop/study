# 뭐든 제대로 이해하는 방법
## 채널 : 이상한 리뷰의 앨리스

### 영상 요약
- 파인만의 교육 방법
    1. 이해하고 싶은 개념 선택하기
    2. 강의용 스크립트를 적기
        - 마치 모르는 사람에게 가르치듯이 스스로에게 설명하는 스크립트 적기
    3. 다시 읽고, 다시 배우기
        - 종이에 적고 설명할 수 있을때까지
    4. 다시 정리하기
- 어떤 개념을 이해해야 한다면 배운 학습을 더 생생한 스토리로 만들어 기억해야한다.
  
### 느낀점
- 이 학습방식은 진짜 다시 해봐야 한다.
